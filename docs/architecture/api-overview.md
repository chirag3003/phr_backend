# API Overview

All authenticated endpoints require:
```
Authorization: Bearer <jwt>
```

Base URL defaults to `http://localhost:5000` when running locally.

## Route Index
- `/auth`
- `/profile`
- `/allergies`
- `/symptoms`
- `/meals`
- `/glucose`
- `/documents`
- `/docDoctors`
- `/uploads`
- `/family`
- `/insights`
- `/water`

See `docs/api/` for detailed request/response examples.
